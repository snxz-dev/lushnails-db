# LUSH NAILS SPA

Sitio web corporativo para la difusión de los servicios de belleza y bienestar de SPA NAILS LUSH. Desarrollado con React 19.

## Requisitos

- Node.js 16+
- npm 8+

## Instalación

```bash
npm install
```

## Scripts disponibles

### `npm start`

Ejecuta la aplicación en modo desarrollo.\
Abre [http://localhost:3000](http://localhost:3000) para verlo en el navegador.

### `npm test`

Ejecuta las pruebas en modo interactivo.

### `npm run build`

Compila la aplicación para producción en la carpeta `build`.

## Estructura del proyecto

```
src/
├── assets/       # Imágenes, videos, iconos
├── styles/       # Archivos CSS
├── App.js        # Componente principal
├── index.js      # Punto de entrada
└── App.test.js   # Pruebas
```

## Metodología

Proyecto desarrollado bajo la metodología en Cascada (Waterfall), con las fases de análisis, diseño, desarrollo, pruebas e implementación.
